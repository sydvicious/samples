### TableMultiSelect ###

===========================================================================
DESCRIPTION:

Demonstrates the use of multiple selection of table cells in UITableView; in particular, using it to delete one or more items.

===========================================================================
BUILD REQUIREMENTS:

iOS SDK 5.0 or later

===========================================================================
RUNTIME REQUIREMENTS:

iOS SDK 5.0 or later
